## 📦 짐 체크리스트

### 의류·기본
- [ ] 기능성 반팔 5~6벌 (땀 흡수 빠른 소재)
- [ ] 반바지 3벌
- [ ] 속옷·양말 충분히 (6~7일치)
- [ ] 얇은 긴팔 1벌 (활옥동굴 12°C, 지리산 저녁)
- [ ] 우비 or 경량 우산 (지리산 필수)
- [ ] 모자 (챙 넓은 것, 선크림 필수 보조)

### 신발
- [ ] **등산화** (지리산 둘레길 2일, 비렁길 필수)
- [ ] 슬리퍼 or 샌들 (숙소·해변)
- [ ] 운동화 (평상시)

### 수상레저용
- [ ] **래쉬가드** (웨이크보드 + 래프팅 필수)
- [ ] **수영복**
- [ ] **방수팩** (핸드폰 보호 — 래프팅·금오도 선착 필수)

### 전자기기
- [ ] 스마트폰
- [ ] **보조배터리** 20,000mAh 이상 (하루 종일 사용)
- [ ] 충전기 + 멀티탭
- [ ] 이어폰

### 세면·위생
- [ ] 세면도구 (숙소 미제공 시 대비)
- [ ] 선크림 SPF50+ (매일 바름)
- [ ] 물티슈
- [ ] 개인 수건 (빠른 건조 소재 추천)

### 비상약 (반드시 챙기기!)
- [ ] **파스** (지리산 2일 후 근육통 확실)
- [ ] 소화제
- [ ] 지사제
- [ ] 해열제·진통제
- [ ] 밴드 + 후시딘
- [ ] 멀미약 (버스 장거리 이동 多)
- [ ] 개인 복용약
